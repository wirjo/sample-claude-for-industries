# Summary of Loyalty Platform Vendor Evaluation

The evaluation compares two loyalty platform vendors for a rewards restructuring initiative.

**LoyaltyCore (Incumbent):** The current provider since 2019 offers minimal integration risk at $2.1M over three years with a 5-month timeline. However, it has significant limitations, particularly in analytics capabilities that would necessitate supplementary reporting tools.

**RewardStack (New Vendor):** This startup alternative costs $2.8M and requires an 8-month implementation window. The recommendation favors RewardStack because it provides "native support for tiers + points + hybrid models" with "built-in real-time dashboards, A/B testing for reward offers, predictive churn modeling."

**Key Justification:** The evaluator argues that RewardStack's comprehensive analytics suite would eliminate the need for maintaining separate reporting infrastructure, offsetting the higher cost through operational savings of approximately $400K annually.

**Next Steps:** Leadership presentation is scheduled for October 28, with procurement targeted to begin by November 15 if approved. The evaluator emphasizes that data residency compliance with Canadian regulations requires confirmation before proceeding.
